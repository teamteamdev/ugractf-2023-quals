Береста

Задача очень проста: нужно вычислить следующую функцию:

f(1) = 1,
f(n) = sum(f(i) | 1 ≤ i < n, n делится на i) + n

Описание ввода-вывода

Изначально на стеке будет храниться только одно значение — аргумент фукции. После выполнения кода на стеке должен остаться только результат.

Пример

Ввод

 5 (top)

Вывод

 6 (top)

На обратной стороне бересты вы нашли инструкцию к ленинецу.
